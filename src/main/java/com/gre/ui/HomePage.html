Welcome to Admon

This simple TODO web application that allows you to do the following.

  • Create, modify and delete tasks
  • Task Collaboration
  • Retrieve task history

All you need to do is register and you can post your tasks. Once you are registered you will be taken to the Task page where you can enter your task.

Important please don't forget the token you entered during registration process.

The token plays an important role in creating or modifying tasks and collaborating with other users. Only those users whom you want to collaborate with can modify your tasks. Do note that this is still a beta release and token feature is not yet available.


On our future releases we are expecting the following features to be available

  • Token Feature
  • Set Reminder
  • Reminder notification via mail
  • Improved task search
  • Improved reporting
